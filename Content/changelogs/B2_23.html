Minecraft Party Beta 2.23


Additions

  • New Advancement: Garbageman (discard 3 items in one turn)
  • Players now have their team prefix before turn order is decided in pairs
  • More helpful signage in Pipe Canyon
  • Star Seller can now stand diagonally
  • A modified version of the crazy clock is now obtainable in Star Race
  • Better lighting in Pipe Canyon pedestal room

Changes

  • Players now place traps from slightly above the board for better visibility
  • Added ladders onto the corners of the ring in Ring in the Ring for better spectation
  • Players are now forcibly dismounted for committing grand theft auto in Raft Racers
  • Ghost Mode now bypasses blocked pipes in Pipe Canyon
  • Buffed cocoa beans in Farming Frenzy (3pts â†’ 4pts)
  • Painting task on PC no longer requires boot wash
  • Reduced PC painting task duration to 30 seconds
  • PC painting task canvas now starts as terracotta, so all spots must be painted
  • Removed movement-based Betris rotation controls (clicking controls still work)
  • Betris blocks can now be rotated using drop and off hand key maps
  • Betris controls in instructions now match player keybinds
  • Updated directions in PC painting task to reflect new rules
  • Port Town Cannon now uses the same system as the updated trap placements
  • Spectators in Pig Pushers are now placed into spectator mode and may spectate the pigs
  • Players are no longer able to drop their "orbs" in Maze Navigator
  • Players now always spawn facing their horse during Horse Race setup
  • Removed melee from Tanks because it was jank
  • Arrow in Tanks no longer respect the laws of physics (delayed hitscan)

Fixes

  • One of the pipes in Pipe Canyon was asymmetrical
  • Some villagers were allowed to exist when players are absent on Pipe Canyon
  • Some paths between spaces on Pipe Canyon would inadvertently activate a battle minigame
  • Players attempting to place a trap would sometimes break their neck and be unable to set the trap
  • Barriers make some spaces impassible on Pipe Canyon
  • Bew does not allow a random player to be selected in Mega Party
  • Pipe Canyon shops would fail to open if you started your turn too close to them
  • There were several bugs with various games in Grand Prix mode
  • Certain Pipe Canyon events did not return the "view board" item to the player
  • Warp star warp count was calculated incorrectly
  • Players were able to trigger spectate during the PC cannon diffusal event
  • Surprise shops did not update the above-board display
  • Some coin pile spawns were permanent on Pipe Canyon
  • Updated One Shot Wonder rules regarding respawn time
  • Some Pipe Canyon warps caused subsequent spaces to go uncounted
  • Some star spawns on Pipe Canyon caused a warping player to get stuck in a block
  • PC mushroom maze would sometimes be impossible
  • Swapping out a player would not update their custom bonus star stat for Pipe Canyon
  • In the Zone was balanced incorrectly
  • Certain star spawns made Pipe Canyon spot detection a little wonky
  • Players were able to leave their pedestals in the Pipe Canyon victory room after being placed

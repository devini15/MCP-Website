Minecraft Party Beta 2.18


Additions

  • Double Bagel advancement for finishing a game with 0 Coins and 0 Stars
  • Shops now display brief item descriptions above signs
  • Cave Parkour now sends a chat message when a player completes the course
  • Horse Race now sends a chat message when a player has completed both laps
  • NEW MINIGAME: Mouse Trap (1v3/2v6)
  • New setting allows shop contents to be hidden until a player approaches the shop

Changes

  • Added some more arrows to Mooshroom Grotto
  • Removed an invalid coin pile spawn location on Port Town
  • Adjusted how placements are calculated in Raft Racers in team mode
  • Removed houses from Meat Slap arena
  • Increased size of text display for remaining trap uses.

Fixes

  • An arrow on Mooshroom Grotto pointed the wrong way
  • Roll Off rolls would have the Minion Pet subtitle if a Minion pet was used on the same turn
  • If a player using a minion pet triggered a Roll Off, the game would softlock
  • Players 5-10 did not get coins from Hungry Hungry Humans
  • Stats book still had outdated bonus star info for Mooshroom Grotto
  • Pairs mode max items setting was not configured correctly
  • Zen Garden's star shuffle did not work correctly when one of the players had no stars.
  • One part of Zen Garden did not allow players to pass normally
  • Protected stars were not counted correctly in Teams/Doubles
  • Two Birds advancement would be granted at inappropriate times
  • Gladiators would incorrectly result in a draw when played with 6 players in Teams
  • Port Town allowed early star purchases
  • Bew would not preserve Spectation preferences when sending players back to the board on Port Town
  • One of the spaces on Floating Islands would sometimes have incorrect behaviour

Minecraft Party Alpha 2.16


Additions

  • Spyglasses for DQd players in Spleef
  • Sound plays when you are passing a quest on Floating Islands
  • Implemented custom board star for Port Town
  • Enabled better spectation (post-mortem) for Bombs Away!
  • Additional shop on Port Town
  • World icon for Minecraft Party 2
  • Game now checks if command blocks are enabled on the server before starting
  • Message in chat when a player finishes all tasks in Tedious Tasks
  • Functionality to swap out players after the original player has left the game
  • Completing the meat quest now tells everyone what item you picked as a reward
  • Port Town now has a new funny magenta spot
  • Added some bonus advice for the hunter in Block Hunt

Changes

  • Multiple Last 5 Turns events now stack
  • Spawnpoints are now updated after completing a game in Quickplay
  • Non-players now have their scores reset after each minigame to prevent them from appearing on future scoreboards
  • Cook meat at speeds never before seen on Floating Islands
  • Slightly improved collision physics in Boat Race
  • Made correct path more obvious at the fork in Pearl Hurl
  • Non-participants in Fight! Fight! Fight! are now put in spectator mode
  • Drink Quest sign now clears upon quest completion
  • Updated stats book to use ':' instead of '-' as a separator
  • Re-enabled Tanks for High Ping mode as it has been optimized

Fixes

  • Players would clip through the balance beam in Balance Beam
  • Spectator names would show up in title text at the end of the game
  • Players could collide with traps
  • Spectators not warped back to board after Port Town
  • Losing items on a black space would not take away MCP2 exclusive items
  • +5 Chance Time space event did not work on Port Town
  • Swap Spot Colors event did not work on Port Town
  • Quickplay does not start properly if spectators are present
  • Custom projectile in Deuce does not get assigned the correct velocity when struck
  • Custom projectiles in Bombs Away! do not move correctly when thrown
  • Players could escape the Deuce arena
  • Villagers stop wanting to do business with you in Tedious Tasks
  • You could get morning wood in Tedious Tasks
  • You could jump very high in Tunnel o' Fun if you get punched just right before the game starts
  • Typo in a quest on Floating Islands
  • Sticky glue would cancel any item effect the player had queued for that turn
  • Item count was not updated correctly when receiving quest rewards
  • Fireballs did not work correctly in Boss Brawl
  • Rotation did not work in Betris
  • The tripwire hook was missing in Port Priority
  • Players are sometimes unable to leave the board spawn on turn 1
  • Items would show up in the wrong order in your hotbar after a special event is triggered

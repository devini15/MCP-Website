Minecraft Party Beta 2.33


Additions

  • Banners in Lost City to make Bew more noticeable

Changes

  • Shortened Lost City music loop
  • Added arrow pointing to Bew on Lost City
  • Coin Thief is now called Pickpocketer
  • Changed Creeper Catchers to Cat Catchers
  • Modified boat physics to make it easier to aim in Tanks
  • Fall damage is no longer enabled in the main hub
  • Non-participants are held in spectator for a bit longer after Go Fish!
  • Space Jumpers now announces a course completion in chat
  • Added light blocks to MG 4fa victory room
  • Dripstone dance moved from normal minigame pool to battle minigame pool

Fixes

  • Build above Bew on Lost City was asymmetrical
  • Block Buster did not update gamemodes correctly in doubles
  • Some super items did not activate when used
  • Coin Shower message has wrong number in chat during Double Trouble
  • Incorrect Music plays during Micro Duels
  • Hardini didn't clean his room when asked
  • Several minor balance/build issues were present in Block Buster
  • Block Buster now always selects the correct version of the game for the number of participants
  • Some 2v2 games would get wrongfully disabled with certain player counts
  • Jumpers were provided with a bow in Space Jumpers
  • Swapping out a player would not update their suffix in the tab list
  • MG altar text displays were facing the wrong way
  • Spirit Shields would not get consumed when they should have
  • Swapped in players would occasionally start their turn in spectator mode
  • Rocket Launch would not display players on the sidebar if they lagged at the wrong time
  • One of the Lost City shops would not close

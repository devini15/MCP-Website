Minecraft Party Alpha 2.2


Additions

  • Added tip to Slime Time informing Slime players that they can jump for extra speed
  • Added enhanced spectation to Go Fish!
  • Added "High Ping" mode that removes or modifies intensive minigames
  • Added d20 item (Big Dice)
  • Option to limit max items a player can hold
  • There are now ghosts in Betris
  • Coin Minigames
  • More bonus star customization options
  • New minigame: Pearl Hurl
  • New minigame: Mineshaft Madness
  • New minigame: Runner
  • New minigame: Market Mayhem

Changes

  • Team of 3 is now put in spectator after dying in Boss Brawl
  • Ghost mode makes players invisable
  • Optimized board function to reduce number of balls chugged per tick
  • Memory Lane now ends automatically
  • Stick now goes into 1st hotbar slot in Rock, Paper, Scissors, Chase
  • You can now actually see what's happening during Chance Time
  • Players now have a wider collision radius in Pac Cube
  • Modified lore of lottery tickets
  • Beacons are no longer present on board during minigames
  • Added spaces before parenthetical text in Bombs Away! instructions
  • Pigstep is no longer too loud at game end
  • Rebalanced scoring for In the Zone to favor the 1 more.
  • Players are now invisible with boots in Parkour
  • Caught players in Block Buster are put into Spectator mode
  • Slime traps now cause you to get carried in Gold Rush
  • Capture the Flag now ends after one capture
  • Capture the Flag respawn delay now increases the longer the game is played
  • Port Priority Beacons now removed while game is not being played
  • Leveled some terrain in Rocket Launch
  • In the Zone is now only on fire while the game is being played
  • Players can no longer move after hide time expires in Block Buster
  • Dead ends no longer appear next to each other in Labyrinth
  • Locked several containers
  • Placement pillars now same height in event of a drawn duel
  • Adjusted price of Star Protector
  • Removed weakness from ghosts in ghost hunt

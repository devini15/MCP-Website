Minecraft Party Beta 2.10


Additions

  • Chat messages when a player completes Raft Racers
  • Cheat detection for Netherite Scrap challenge on Floating Islands
  • Decorative Island on Floating Islands
  • Extra code to make it more difficult to cheat over diagonals in Tile Capture
  • Barriers to prevent people from falling off the Floating Islands victory platforms
  • AustinTheManiac added to media section
  • NEW GAMEMODE: MEGA PARTY MODE

Changes

  • Failsafe spawnpoint is now set when starting the Zen Garden disc challenge
  • Modified how the bosse's hotbar is managed in Boss Brawl to prevent item loss
  • Swapping stars at Chance Time is now slightly more likely in solos
  • Double Star Voucher won from trick or treat now has a 50% chance of being a Minion Pet instead
  • Port Town whirlpool now has flowing water
  • Changed default balance of Lost City spaces (-1 Duel Space, +1 Blue Space)
  • Removed upper bound from Floating Island's Sacrificial Altar
  • Reduced max coins stealable with Port Town cannon
  • Increased AOE of Port Town cannon
  • There are now stairs leading up to the Meat Slap bridge making it much less beneficial to camp there
  • Communism now redistributes all coins even if they cannot be distributed perfectly evenly

Fixes

  • Some Coin Minigame platform colors would overlap oddly
  • Invalid star spawn on Zen Garden
  • Coin Thief icon was visible to early
  • Player spots continued to glow during Coin Minigames
  • Canceling the game while a Ghost Mode is active caused 3 to get added to the turn-order dice in the next game
  • Non-players were still assigned doubles teams
  • Typo in Fire Trap description
  • Free item pickups had buggy behaviour in Port Town
  • Tunnel o' Fun would sometimes generate outside the intended region and partially fail to get removed after the game
  • Players could roam freely in market mayhem before the game started
  • Players had a 3-second window to kill each other while Bat Blast was loading
  • Some board-specific entities would not spawn on the 1st turn
  • Sacrificial altar did not allow sacrifices in excess of 64 Coins
  • Swapping positions with someone on the lucky path in Port Town would still award a free star but you'd also be stuck there forever and softlock the game so it was hardly worth it

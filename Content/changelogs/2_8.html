Minecraft Party 2.8


Additions

  • Spectators receive list of items that players are searching for in Port Priority
  • Players now get launched on Port Town when entering the Whirl Pool

Changes

  • Quickplay mode now auto-changes to solos mode if pairs mode was selected upon setup
  • Superstar Equipment (The best armor set) now uses amethyst trim instead of gold
  • Removed a between space on Mooshroom Grotto that was immediately after a free item pickup
  • Board effects are now conditional for players that are not actively in a challenge
  • Slightly reworked win condition for large team in Mouse Trap so that mice have time to escape a 1x1 using their axes at the last few seconds of the minigame
  • Increased waypoint detection area in Take Flight
  • Updated welcome book
  • Increased team of 3's speed in 2v3 Slime Time after the 1st gold player gets eliminated
  • On Rhythmic Room, players now get teleported back to the turn player after star cost changes from a player buying a Star

Fixes

  • Player could get stuck by free item pickup on Port Town
  • A lingering variable that keeps track of players that finished Puzzled did not get reset after the minigame ended causing errors in other minigames
  • Player scores could create "gaps" in quickplay mode if a player left thus causing issues on various minigames
  • Error in Ghast Blast rules
  • Summon Single-Use Star Seller Special Event could spawn the star sellers in spaces occupied by equipment
  • Missing space character when approaching star seller on Arcane Atoll
  • Right-clicking 'Go To Spectate Mode' item could put other players into spectator mode
  • Memory Lane jump scoreboard would disappear and not get added back into the game
  • Freeze Trap did not swap between players during the chance time swap items event
  • "The magic has run out" cave task message could appear twice if the previous player returned to the Miner before the magic ran out on MG
  • Thorned Spirit Shields did not update in inventory when players return to the board
  • Extra space characters in the subtitles for extra roll bonuses
  • Inconsistent bolding in tellraws when players win items from the sacrificial altar
  • Losing all items did not remove a player's Double Star Voucher
  • Speed/Jump Boots did not work in Gold Rush
  • Gold Rush had random underlined words when listing points from each team
  • MPIQ 'Speedy' equipment color was shown in dark blue instead of aqua
  • Removed the extra code that was giving LordODragon bad luck
